# Stock Market Prediction Web Application

A machine learning based stock market prediction web application built using Python and Flask.
This project analyzes historical stock data and predicts future stock price trends.

---

## Features
- Stock price prediction using machine learning
- Separate model training and prediction modules
- Web application using Flask
- Simple and clean user interface

---

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS

---

## Project Structure
backend/             # Backend logic (if any)
app.py               # Main Flask application
model_train.py       # Model training logic
model_predict.py     # Prediction logic
templates/           # HTML files
static/              # CSS files

---

## How to Run the Project

1. Install dependencies
pip install -r requirements.txt

2. Run the application
python app.py

3. Open in browser
http://127.0.0.1:5000/
